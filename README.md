# Keep Thought - 前端技术示例代码集合

这是一个收集各种前端技术示例代码的仓库，用于学习和参考。

## 📁 项目结构

```
keep-thought/
├── README.md                 # 本文件
├── react19-demos/           # React 19 新特性演示
│   ├── src/demos/           # 各种 React 19 示例
│   ├── README.md            # React 19 项目说明
│   └── package.json         # 项目依赖
│
├── redux-toolkit-demo/      # (待添加) Redux Toolkit 示例
├── nextjs-demo/             # (待添加) Next.js 示例
└── typescript-demo/         # (待添加) TypeScript 高级用法
```

## 🚀 已包含的示例

### 1. React 19 Demos
React 19 新特性完整演示，包含：
- **Actions** - 异步操作新范式
- **useOptimistic** - 乐观更新
- **use() API** - 资源读取
- **ref as prop** - 无需 forwardRef
- **Form Actions** - 表单操作简化
- **Metadata** - 文档元数据支持

[查看详情](./react19-demos/README.md)

## 🛠️ 如何使用

每个子目录都是一个独立的项目，可以单独运行：

```bash
# 进入某个示例目录
cd react19-demos

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
