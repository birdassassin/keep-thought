# Next.js 15 完整教程

## 版本信息
- Next.js: 15.x
- React: 19.x
- Node.js: 18.17+

## 核心特性
- App Router（文件系统路由）
- Server Components（默认服务端渲染）
- Server Actions（服务端操作）
- Turbopack（新一代打包工具）

## 快速开始

```bash
# 创建新项目
npx create-next-app@latest my-app --typescript --tailwind --eslint --app

# 进入项目目录
cd my-app

# 启动开发服务器
npm run dev
```

## 学习路径

### 第一阶段：基础入门
1. 了解 Next.js 是什么，为什么选择它
2. 掌握项目创建和目录结构
3. 理解 App Router 的基本概念

### 第二阶段：核心功能
1. 深入理解 Server Components 和 Client Components
2. 掌握数据获取和缓存策略
3. 学习 Server Actions 处理表单和数据变更

### 第三阶段：进阶应用
1. 中间件和路由拦截
2. API Routes 开发
3. SEO 和性能优化
