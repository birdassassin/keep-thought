# ES6+ 现代 JavaScript 完整教程

## 版本信息
- ES6 (2015) - ES2024

## 核心特性
- 箭头函数、解构赋值、模板字符串
- Promise、async/await
- 模块化、类、Symbol
- 迭代器、生成器
- 新增数组/对象方法

## 示例目录
1. 01-basics.js - 基础语法
2. 02-functions.js - 函数进阶
3. 03-objects.js - 对象与解构
4. 04-async.js - 异步编程
5. 05-modules.js - 模块化
6. 06-advanced.js - 高级特性

## 如何运行示例

```bash
# 使用 Node.js 运行单个文件
node src/examples/01-basics.js

# 或者在浏览器开发者工具中直接运行代码片段
```

## 学习建议

1. **循序渐进**：按照 01-06 的顺序学习，每个文件都建立在前一个的基础上
2. **动手实践**：不要只看代码，要在浏览器控制台或 Node.js 中实际运行
3. **修改实验**：尝试修改示例代码，观察不同的输出结果
4. **查阅文档**：遇到不理解的地方，查阅 MDN Web Docs 获取详细说明

## ES6+ 版本演进

| 版本 | 年份 | 重要特性 |
|------|------|----------|
| ES6 | 2015 | let/const、箭头函数、类、模块、Promise、Symbol、迭代器、生成器 |
| ES7 | 2016 | includes()、指数运算符(**) |
| ES8 | 2017 | async/await、Object.entries()、Object.values()、字符串填充 |
| ES9 | 2018 | rest/spread属性、异步迭代、Promise.finally() |
| ES10 | 2019 | Array.flat()、Object.fromEntries()、String.trimStart/End() |
| ES11 | 2020 | 可选链(?.)、空值合并(??)、BigInt、动态导入 |
| ES12 | 2021 | 逻辑赋值运算符、数字分隔符、String.replaceAll() |
| ES13 | 2022 | 顶层await、类私有字段、.at()方法 |
| ES14 | 2023 | Array.fromAsync()、Hashbang语法 |
| ES15 | 2024 | 顶层await优化、Promise.withResolvers() |
